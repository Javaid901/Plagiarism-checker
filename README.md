# AI-Powered Plagiarism Checker

## Overview

The AI-Powered Plagiarism Checker is a web application developed using Python and Flask to detect textual similarity between documents. The system compares uploaded text with stored documents and generates a plagiarism percentage along with matching content.

## Features

- Upload text files for plagiarism detection
- Calculate similarity percentage
- Display matching content
- Simple and responsive web interface
- Fast plagiarism analysis

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- Jinja2

## Installation

```bash
git clone https://github.com/Javaid901/Plagiarism-checker.git
cd Plagiarism-checker
pip install -r requirements.txt
python app.py
```

Open:

```
http://127.0.0.1:5000
```
